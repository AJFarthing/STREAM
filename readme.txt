

The contents of this folder are as follows :-

-----------------------------------------------------------------------------------------------------------------------------------------------
Readme.txt : This readme file.
-----------------------------------------------------------------------------------------------------------------------------------------------
setup.sh : This script sets up a Streamlit configuration file with specific options for the Streamlit server, ensuring that it runs in headless mode, sets the appropriate port for deployment, and disables CORS restrictions. It's typically used when deploying a Streamlit app to a hosting platform.
-----------------------------------------------------------------------------------------------------------------------------------------------
main.py : This is the script for the StreamLit app.
-----------------------------------------------------------------------------------------------------------------------------------------------
lgb_model.pkl : This is the model used in the script for making crop recommendations in the app.
-----------------------------------------------------------------------------------------------------------------------------------------------
requirements.txt : This is a list of packages needed to run the app, it's only three lines.
-----------------------------------------------------------------------------------------------------------------------------------------------
Crop_recommendation.csv : This is the csv file storing data for modelling
-----------------------------------------------------------------------------------------------------------------------------------------------
CROPS.ipynb : This is a Jupyter Notebook containing brief EDA of the Crop_recommendation.csv file as well as model training for the lgb_model.pkl file.
